DATETIME.TIMEDELTA — FEATURES AND LOGIC

1. What is timedelta?
- A timedelta object represents a duration, i.e., the difference between two datetime or date instances.
- It can express time differences down to microsecond resolution.

2. Constructor
class datetime.timedelta(days=0, seconds=0, microseconds=0,
                         milliseconds=0, minutes=0, hours=0, weeks=0)

- All arguments are optional and default to 0.
- Arguments can be integers or floats, positive or negative.
- Internally, only days, seconds, and microseconds are stored.
- Other units are converted:
  - 1 millisecond = 1000 microseconds
  - 1 minute = 60 seconds
  - 1 hour = 3600 seconds
  - 1 week = 7 days

3. Normalization
- Values are normalized so that:
  - 0 <= microseconds < 1,000,000
  - 0 <= seconds < 86,400 (number of seconds in a day)
- Example:
    from datetime import timedelta
    delta = timedelta(days=50, seconds=27, microseconds=10,
                      milliseconds=29000, minutes=5, hours=8, weeks=2)
    print(delta)
    # Result: timedelta(days=64, seconds=29156, microseconds=10)

4. Attributes
- timedelta.days → number of days
- timedelta.seconds → number of seconds (0–86,399)
- timedelta.microseconds → number of microseconds (0–999,999)

5. Class Attributes
- timedelta.min → most negative timedelta (-999999999 days)
- timedelta.max → most positive timedelta (999999999 days, 23:59:59.999999)
- timedelta.resolution → smallest possible difference (1 microsecond)

6. Methods
- total_seconds() → returns the total duration expressed in seconds (float).

7. Supported Operations
- Addition/Subtraction:
    t1 = t2 + t3
    t1 = t2 - t3
- Multiplication/Division:
    t1 = t2 * i   # integer
    t1 = t2 * f   # float
    t1 = t2 / i   # division
- Floor division, modulo, divmod also supported.
- Negation and absolute value:
    -t1, abs(t1)
- Comparisons:
    timedelta objects support equality and ordering.
- Boolean context:
    A timedelta is True if it is not equal to timedelta(0).

8. Examples
from datetime import timedelta

year = timedelta(days=365)
ten_years = 10 * year
print(ten_years)  # timedelta(days=3650)

nine_years = ten_years - year
print(nine_years)  # timedelta(days=3285)

three_years = nine_years // 3
print(three_years)  # timedelta(days=1095)

print(year.total_seconds())  # 31536000.0

9. Usage
- timedelta is commonly used to:
  - Calculate differences between dates/times.
  - Add or subtract durations from datetime objects.
  - Represent intervals in scheduling, expiration, or countdown logic.

SUMMARY
- timedelta represents a duration between two dates/times.
- It supports arithmetic, comparisons, and conversion to seconds.
- It is essential for date/time calculations in Python.
