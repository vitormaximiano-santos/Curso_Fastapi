Function: pegar_sessao()
# dependency for the database session
def pegar_sessao():
    try:
        session = sessionmaker(bind=db) # bind é o objeto que armazena a conexão com o banco de dados
        session = session()
        yield session
    finally:    
        session.close()

Explanation:
  - "sessionmaker(bind=db)": creates a session factory bound to the database connection object (db).
        - "sessionmaker":The sessionmaker factory generates new Session objects when called, creating them given the configurational arguments established here.
        - "bind=db": An optional Engine or Connection to which this Session should be bound. When specified, all SQL operations performed by this session will execute via this connectable, in this case, the connection is "db" imported from the "models" file.

  - "session = session()": initializes a new session instance.
        - "session": this is a variable that stores the session object, which is created by the sessionmaker function.
        - "session()": create a new session object, it is usage for operations with the database (add, update, delete, etc.)
           - example: session.add(new_user)

  - "yield session": returns the session to the caller while keeping the function open 
    (generator behavior).
  - "finally: session.close()": ensures the session is closed after use, preventing leaks.

Key Concepts:
  - "Session": temporary connection to the database used for queries and transactions.
  - "Dependency Injection": FastAPI can inject this session into routes automatically.
  - "Yield": Python keyword that allows generator functions to return values without ending execution.

Purpose/Function in the code:
  Provides a reusable database session dependency for FastAPI routes, ensuring 
  safe connection handling and automatic cleanup after each request.

# verifies the validity of a JWT token and retrieves the corresponding user

def verificar_token(token:str = Depends(oauth2_scheme), session:Session = Depends(pegar_sessao)):
    try:
        dic_info  = jwt.decode(token, SECRET_KEY, ALGORITHM)
        id_usuario = int(dic_info.get("sub"))
        
    except JWTError:
         raise HTTPException(status_code=400, detail="Acesso negado,verifique a validade do token")
    
    usuario= session.query(Usuario).filter(Usuario.id == id_usuario).first()
             
    if not usuario:
        raise HTTPException(status_code=401, detail="Acesso negado, usuário não encontrado")
    return usuario

        - "token:str": get a token created in "oauth2_scheme", and convert it to string
