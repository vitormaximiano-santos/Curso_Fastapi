Bibliotecas do curso FastAPI - Hashtag Treinamentos

===========================
Segurança e Criptografia
===========================

bcrypt
Propósito: Hash de senhas
Utilidade: Criptografar senhas de forma segura.
Termo-chave: "hash" = transformação irreversível usada para autenticação.

cryptography
Propósito: Criptografia geral
Utilidade: Implementa algoritmos de segurança (AES, RSA, etc.).

ecdsa
Propósito: Assinaturas digitais
Utilidade: Criptografia baseada em curvas elípticas.

passlib
Propósito: Hash de senhas
Utilidade: Biblioteca robusta para gerenciamento de senhas.

pyasn1
Propósito: ASN.1
Utilidade: Manipulação de estruturas de dados criptográficas.

rsa
Propósito: Criptografia RSA
Utilidade: Implementa algoritmo RSA para chaves e assinaturas.

python-jose
Propósito: JWT e criptografia
Utilidade: Implementa JSON Web Tokens e algoritmos JOSE.

===========================
Web e Servidor
===========================

fastapi
Propósito: Framework web
Utilidade: Framework moderno e rápido para APIs em Python.

starlette
Propósito: Framework ASGI
Utilidade: Base do FastAPI, fornece roteamento e middlewares.

uvicorn
Propósito: Servidor ASGI
Utilidade: Servidor rápido para rodar aplicações FastAPI.

h11
Propósito: HTTP/1.1
Utilidade: Implementação pura de HTTP/1.1 usada pelo Uvicorn.

python-multipart
Propósito: Upload de arquivos
Utilidade: Suporte para multipart/form-data.

idna
Propósito: Domínios internacionais
Utilidade: Suporte a nomes de domínio com caracteres especiais.

===========================
Banco de Dados
===========================

SQLAlchemy
Propósito: ORM
Utilidade: Manipulação de bancos relacionais.

SQLAlchemy-Utils
Propósito: Extensões para SQLAlchemy
Utilidade: Tipos adicionais e utilidades.

Alembic
Propósito: Migrações de banco de dados
Utilidade: Gerenciar versões e alterações de esquema.

===========================
Validação e Tipagem
===========================

pydantic
Propósito: Validação de dados
Utilidade: Valida e serializa dados com base em modelos.

pydantic_core
Propósito: Núcleo do Pydantic
Utilidade: Motor interno de validação.

annotated-types
Propósito: Tipagem avançada
Utilidade: Criação de tipos anotados para validações precisas.

typing-inspection
Propósito: Tipagem
Utilidade: Inspecionar tipos em tempo de execução.

typing_extensions
Propósito: Tipos extras
Utilidade: Tipos novos para versões antigas do Python.

===========================
Utilidades e Ferramentas
===========================

anyio
Propósito: Concorrência assíncrona
Utilidade: Abstrai asyncio, trio e curio.

greenlet
Propósito: Corrotinas leves
Utilidade: Suporte para multitarefa cooperativa.

sniffio
Propósito: Detecção de loop assíncrono
Utilidade: Identifica qual biblioteca assíncrona está em uso.

cffi
Propósito: Interface C
Utilidade: Permite chamar código C dentro do Python.

pycparser
Propósito: Parser C
Utilidade: Necessário para bibliotecas que usam CFFI.

click
Propósito: CLI
Utilidade: Facilita criação de comandos de terminal.

colorama
Propósito: Cores no terminal
Utilidade: Adiciona cores e estilos em saídas de console.

python-dotenv
Propósito: Variáveis de ambiente
Utilidade: Carrega configurações de arquivos .env.

six
Propósito: Compatibilidade
Utilidade: Suporte entre Python 2 e 3.

Mako
Propósito: Templates
Utilidade: Sistema de templates usado pelo Alembic.

MarkupSafe
Propósito: Segurança em templates
Utilidade: Escapa caracteres perigosos em HTML/Jinja/Mako.
