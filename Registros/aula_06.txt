Arquivo: rotas_autenticacao.py
Funcionalidade:
- Criação de rotas relacionadas à autenticação de usuários.
- Implementação de login com geração de tokens JWT.
- Implementação de refresh token para renovar tokens expirados.

Funções principais:
1. criar_token(id_usuario, duracao_token):
   - Gera token JWT com informações do usuário.
   - Define data de expiração com base em `ACCESS_TOKEN_EXPIRE_MINUTES`.
   - Usa `jwt.encode()` para criar o token.

2. verificar_token(token):
   - Função para validar token recebido.
   - Extrai informações do usuário a partir do token.
   - Retorna usuário associado.

3. autenticar_usuario(email, senha, session):
   - Verifica se usuário existe no banco.
   - Valida senha criptografada com bcrypt.
   - Retorna objeto usuário se válido.

Rotas:
1. GET /auth/
   - Rota padrão de autenticação.
   - Retorna mensagem de acesso.

2. POST /auth/Criar_Conta
   - Cria novo usuário no banco.
   - Criptografa senha antes de salvar.
   - Retorna mensagem de sucesso.

3. POST /auth/Login
   - Recebe email e senha.
   - Autentica usuário.
   - Gera `access_token` com expiração curta.
   - Gera `refresh_token` com expiração maior (7 dias).
   - Retorna ambos os tokens.

4. POST /auth/Refresh_Token
   - Recebe refresh token.
   - Valida token.
   - Gera novo access token.
   - Retorna token atualizado.

Observação:
- Uso da biblioteca `python-jose` para manipulação de JWT.
- `datetime` e `timedelta` usados para controlar expiração dos tokens.
- Agora o sistema tem suporte a **login seguro com JWT** e **refresh token**.
